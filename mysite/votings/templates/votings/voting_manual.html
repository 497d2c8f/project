{% endblock h1 %} {% block content %}

Если Вы в роли Голосующего:

1. Открыть страницу голосования с открытой регистрацией и выбрать посредника.
2. Дождаться закрытия регистрации.
3. В программе выбрать режим "Голосующий", загрузить в программу файл голосования скачанный со страницы голосования и проголосовать.
4. Загрузить созданный программой файл на страницу голосования.
5. Ждать итогов голосования.
6. При желании скачать со страницы голосования файл с его результатами для их проверки программой.

Если Вы в роли Посредника:

1. Открыть страницу голосования с открытой регистрацией и стать посредником.
2. Дождаться закрытия регистрации и сообщений выбравших Вас голосующих.
3. Скачать файл msg2_wm и файл голосования.
4. Скачать файл голосования со страницы голосования.
5. В программе в режиме "Агенство" пройти финальный этап голосования, следуя инструкциям программы.
6. Загрузить файл "s_a_vres__*" на страницу голосования, для отображения его итогов.

Если Вы в роли Агенства:

1. Открыть регистрацию участников в голосовании.
2. Закрыть регистрацию по прошествии необходимого Вам времени.
3. Дождаться от посредников сообщений со ссылками на файл "s_m_msg4_ma__*" и скачать их в одну папку.
4. Скачать файл голосования со страницы голосования.
5. В программе в режиме "Агенство" пройти финальный этап голосования, следуя инструкциям программы.
6. Загрузить файл "s_a_vres__*" на страницу голосования, для отображения его итогов.

После этих действий Вам будут доступны создание голосований и участие в них.
