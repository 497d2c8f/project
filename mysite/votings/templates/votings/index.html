Чтобы начать пользоваться проектом, необходимо:

1. Скачать программу с сайта.
2. В программе создать файл с ключами. Этот файл нужно хранить в секрете. ВАЖНО! В случае утери файла с ключами будет утерян доступ ко всему, что с этими ключами было связано.
3. В программе воспользоваться функцией создания одноразового ключа для регистрации/входа при помощи специального кода на сайте.
4. В программе скопировать публичные ключи и загрузить их в свой профиль на сайте.

После этих действий Вам будут доступны создание голосований и участие в них.
